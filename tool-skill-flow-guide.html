      调用时机矩阵

      机制 函数 何时调用 为什么在这里
      Skill 层装配 ContextBuilder._build_skill_layer 每轮构建 system prompt 时 把 always skill 全文和 available skill 摘要注入 prompt。
      Tool Schema 暴露 TurnProcessor._filter_tool_definitions 每次 LLM 请求前 按 lane 过滤可见工具，例如 `main` 隐藏 `message`。
      Tool Guard 执行 ToolExecutionGuard.execute 每个 tool_call 先做 deferred/handler 判定，再进入 validate + execute 管线。
      结果回填 ContextBuilder.add_tool_result 每个 tool_call 之后 让模型读取 tool 输出进入下一轮推理。
